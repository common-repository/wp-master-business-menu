== Description ==

WP Master Business Menu allows you to create a simple and modern looking menu. This is great for displaying restaurant menus, venue events and much more. You can create multiple menus and display any amount of menus on a single page.

After publishing your menu, simply copy and paste the shortcode into any page or post. You'll find this is a very easy way to keep track of your menu and even easier to update.

Please view the demo here: http://wpmaster.com/plugins/wp-master-business-menu/

== Installation ==

1. Upload 'wpm-business-menu' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Press "Add New Business Menu" to create a menu. Save it once you're done.
3. Simply copy the shortcode into any page or post.
